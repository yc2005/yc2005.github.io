📢 下单提醒怎么收？

对方下单时，提醒会通过 钉钉（和微信）自动发出，关掉 App 也能收到 —— 比浏览器通知靠谱多了，不用在这里点任何开关。

想收到提醒，确认这几点 👇

  1. 你和对方都加入了你们的钉钉群（群里的机器人负责发"对方下单啦"）。
  2. 手机里钉钉的通知是开着的。
  3. 这个钉钉群没有被设成"免打扰"。

💡 技术配置（GitHub 那套）已经一次性设好了，平时不用动。对方一下单，钉钉群里马上就有提醒。

🔑 设置同步 Token

为了让回忆在不同设备间同步，需要一个 GitHub 访问令牌。它只保存在当前这台设备的浏览器里，不会写进网页或仓库，因此不会被 GitHub 自动作废。换设备时在新设备上再填一次即可。

怎么获取 Token？（点击展开）
  1. 登录 GitHub → 右上角头像 → Settings
  2. 左侧最下 Developer settings → Personal access tokens
  3. 新建 Token，授予对仓库 yc2005.github.io 的 Issues / repo 读写权限
  4. 复制并粘贴到上方输入框

⚠️ Token 只显示一次，请妥善保存。
